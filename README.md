# webscraper-indeed
un webscraper programmé pour le site indeed, qui se charge d'afficher le titre, l'entreprise, le département et la date des 16 premières offres.
installer le packages avec 
```
pip selenium
```
attention le chromedriver (le chrome de selenium) doit être dans le même dossier
si par exemple vous voulez le lancer pour un poste d'architecte voici les étapes :
taper architecte sur indeed 
récupérer l'url de votre navigateur
le rentrer dans la ligne 11
et lancer le programme
